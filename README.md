# Linux Automation Scripts

## Español

Colección de scripts en **Bash** diseñados para automatizar tareas comunes de administración en sistemas Linux.

### Scripts incluidos

- `update-system.sh` → Actualiza el sistema
- `system-info.sh` → Muestra información del sistema
- `backup-home.sh` → Realiza respaldo del directorio home

### Tecnologías

- Bash
- Linux
- Automatización de sistemas

### Objetivo

Facilitar tareas repetitivas de administración del sistema mediante scripts simples y eficientes.

---

## English

Collection of **Bash scripts** designed to automate common administrative tasks in Linux systems.

### Included Scripts

- `update-system.sh` → System update automation
- `system-info.sh` → Displays system information
- `backup-home.sh` → Creates a backup of the home directory

### Technologies

- Bash
- Linux
- System automation

### Purpose

Simplify repetitive system administration tasks using simple and efficient scripts.
